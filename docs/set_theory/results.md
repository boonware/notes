### Power Set

The power set $\mathcal{P}$ of a set $S$ is the set of all subsets. The notation $2^{S}$ is sometimes used because $|\mathcal{P}| = 2^{|S|}$.

#### Proof
Let $|S| = n$

$$
    \begin{align*}
        {n \choose 0} + {n \choose 1} + {n \choose 2} + \dots + {n \choose n} \\
        \sum_{k=0}^{n} \frac{n!}{k!(n-k)!} \\
    \end{align*}
$$
